# Credit Risk Portfolio Simulation (Excel)

## Overview
This project implements a credit risk portfolio model in Excel for multiple firms.

The model estimates potential losses using Monte Carlo simulation based on key risk parameters.

## Inputs
- Probability of Default (PD)
- Loss Given Default (LGD)
- Exposure at Default (EAD)
- Threshold model (default trigger)
- Number of firms: 5

## Methodology
- Simulation of default events using threshold approach
- 500 Monte Carlo scenarios
- Loss calculation at portfolio level

## Outputs
- Expected Loss (EL)
- Value at Risk (VaR 95%)
- Value at Risk (VaR 99%)

## Use Case
Portfolio credit risk assessment and loss distribution analysis.

## Tools
- Microsoft Excel

## Notes
This is a simplified model designed to demonstrate core credit risk concepts used in practice.
